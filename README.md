# miraiLABO-blog
技術ブログgoで制作
## 機能
- マークダウン記法
- 新規作成・消去・編集機能
- Basic認証
- DBで管理

## 開発方法

### 開発環境やツール
- mac
- vscode
- Sequel Pro（DB GUI クライアント）
- heroku
- Go
- mysql
- envrc

### 利用したライブラリ
- echo
- pongo2
- sqlx
- goose
